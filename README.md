# 🔬 젠스파크 AI 소재 분석 시스템

F&F Sergio Tacchini Planning Team

## 📖 소개

실시간 이미지 분석 기반 소재 물성 측정 시스템

### 주요 기능
- 5장 이미지 업로드 (정면/측면/확대/드레이프/뒷면)
- AI 물성 분석 (밀도, 광택, 거칠기, 중량, 두께, 촉감)
- 전문가 피드백 시스템
- 측정 히스토리 관리
- JSON 데이터 Export

## 🚀 사용 방법

### 온라인 버전
배포된 링크로 접속하여 바로 사용

### 로컬 실행
```bash
pip install -r requirements.txt
streamlit run genspark_material_analyzer.py
```
